## Задача 3

### Доработаем классы с компьютерами из предыдущего домашнего задания
Теперь:
- метод **turnOn** принимает текущее напряжение в сети, если напряжение меньше требуемого, то компьютер не включается и выкидывает проверяемую ошибку **ComputerTurnOnException**
- метод **turnOff** успешно отработает, только если компьютер был до этого включен, иначе выкидывает проверяемую ошибку **ComputerTurnOffException**
- метод **connect** принимает статус сети и успешно отработает, только если компьютер был до этого включен, иначе выкидывает непроверяемую ошибку **ComputerConnectionError**

Пусть будут следующие рабочие напряжения:
- персональный компьютер (ПК, он же desctop) - 220В
- ноутбук - 20В
- нетбук - 5В
- моноблок - 220В
- сервер - 380В
- неттоп - 12В

Как их проверять - на ваше усмотрение

Для того, чтобы это осуществить, вам нужно:
1. поменять везде (начать с интерфейса) сигнатуру вызова (т.е. передаваемые параметры в данном случае) методов **turnOn(int currentVoltage)** и **connect(ConnectionStatus status)**, где **ConnectionStatus** - [enum](https://javarush.ru/groups/posts/1963-kak-ispoljhzovatjh-klass-enum)(перечисление). Для методов, где нужно, добавить **throws ...** (это так же должно быть в интерфейсе и во всех наследующих классах)
2. завести соответствующие классы для ошибок
3. в методе **turnOn** проверять, хватает ли напряжения для включения
4. завести флаг **boolean isOn**, показывающий, включен ли компьютер
5. в методе **turnOff** проверять, включен ли компьютер
6. в методе **connect** проверять, включен ли компьютер и работает ли интернет


Про метод main:
- В методе main вам нужно создать экземпляры ваших классов и по очереди для них вызвать в цикле **turnOn, connect, turnOff**
- Пусть для всех будет **одно** напряжение, какое - выбираете сами. Какие-то компьютеры запустятся, какие-то нет, этого мы и хотим
- Ошибки нужно отлавливать в **try-catch**, причем **catch** должно быть **2**, под разные типы проверяемых ошибок
- **ConnectionStatus** пусть всегда будет **OK**

#### Разбалловка:
1. Реализованы классы ошибок - 1б
2. Правильно переделаны интерфес и классы - 1 б
3. Правильно написаны методы - 1б
4. Правильно написан main, где обрабатываются ошибки - 1 б
